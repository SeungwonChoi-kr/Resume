### **컴퓨터 그래픽스 엔진 강의 수강** (2024.12 - 2025.06)

3D 그래픽스 렌더링 파이프라인의 핵심 원리를 학습하고 엔진의 주요 기능을 직접 구현

- 사용 기술: OpenGL, DirectX11, C++
- 주요 학습 및 구현 내용:
    - 렌더링 파이프라인 구현:
        - 그래픽스 파이프라인의 핵심인 래스터화와 레이 트레이싱 알고리즘을 직접 구현하며 빛의 반사, 굴절 원리를 학습
    - 쉐이더 프로그래밍 및 물리 적용:
        - GLSL/HLSL 쉐이더 코드를 작성하여 불꽃, 물결 효과 등 시각적 후처리를 구현
        - 충돌 감지 시스템을 구현하여 렌더링된 객체에 물리 시스템을 적용
    - GUI로 엔진 사용성 향상:
        - 조명, 카메라, 렌더링 옵션을 실시간으로 제어할 수 있는 GUI를 구현
